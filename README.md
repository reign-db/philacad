# The Phil-Cad Philippine Academician

Professional academic publishing website for **The Phil-Cad Philippine Academician** — International Quarterly Magazine.

## Tech Stack

- **React** (Vite)
- **Tailwind CSS**
- **react-router-dom**
- **lucide-react**
- **framer-motion**

## Setup

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — Start development server
- `npm run build` — Production build
- `npm run preview` — Preview production build

## Structure

- `src/components/` — Layout (Navbar, Footer), Home section components
- `src/pages/` — Home, Publication, Journal, Events
- Routes: `/`, `/publication`, `/journal`, `/events`
